# LoHo NFT-Marketplace
LoHo is a decentralized marketplace for NFTs. It is built using Ethereum smart contracts.

# Technologies Used
- Solidity
- Truffle
- Ganache
- Polygon Network
- Rinkeby Network
- React
- Infura API
